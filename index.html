Website Idea (Chicken Shop)

Boppys Chicken Shop is a fully responsive, multi-page e-commerce style website created as a portfolio project to demonstrate modern front-end design and interactivity. Built using HTML, CSS, and JavaScript, the site showcases a fictional chicken restaurant brand with a fully functional cart system, dynamic menu search and filter, day/night mode toggle, and a hidden “secret menu” Easter egg for users to discover. It features a responsive hero image slideshow, an interactive merch carousel powered by Swiper.js, and multiple themed pages for a complete browsing experience. Every component—from the menu layout to the live item counter—was developed with usability and responsive design in mind, ensuring a seamless experience across all screen sizes. Deployed via GitHub Pages, Boppy’s Chicken Shop highlights both creative branding and technical execution, combining playful design with real-world e-commerce functionality.

Don't Touch The Button

Don’t Touch the Button is a cheeky little web game I built using React, HTML, CSS, and JavaScript — and yes, it’s exactly what it sounds like. The goal? Simple: don’t touch the button… but good luck resisting the urge. I designed it to test both curiosity and self-control, wrapping mischievous gameplay in clean visuals and smooth animations. Behind the fun lies a neatly structured React app that handles state, interactivity, and playful logic with flair. It’s a lighthearted showcase of my front-end development skills — where code meets comedy and temptation wins every time.
